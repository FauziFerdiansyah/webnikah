        Format File

        Upload file CSV atau Excel (.xlsx) dengan format kolom berikut:

        1. Nama Tamu - Nama lengkap tamu (wajib)
        2. Jumlah Tamu - Angka (1-10)
        3. Nomor Handphone - Format: +62xxx atau 08xxx (opsional, auto-convert)
        4. Instagram - Username tanpa @ (opsional, @ akan dihapus otomatis)
        5. Tamu Undangan Dari - CPP/CPW/Alfira/Fauzi/dll
        6. Keterangan Undangan Dari - Keterangan tambahan
        7. Table - VIP atau Reguler
        8. Souvenir - VIP atau Reguler

        Catatan: Phone & Instagram keduanya opsional. Jika tidak diisi, button WhatsApp/Instagram tidak akan muncul.

        Format: CSV atau Excel (.xlsx, .xls)
